## 📌 Description

Frontend d’une application **E-commerce** développé avec **React**, permettant aux utilisateurs de :

- 🛍️ Parcourir les produits  
- 🧺 Gérer le panier  
- 📦 Passer des commandes  
- 👤 Gérer leur compte  
- 🌍 Changer de langue (Français / Arabe)  

Cette interface communique avec une API backend Laravel.

---

## 🎯 Objectifs

- Offrir une interface utilisateur moderne et responsive  
- Permettre la navigation fluide entre les pages  
- Gérer les interactions utilisateur (panier, commandes)  
- Supporter le multilangue (FR / AR)  
- Consommer les API REST  

---

## 🧱 Architecture

L’application est structurée en composants React réutilisables.

### 🔄 Workflow

Utilisateur → Interface React → API Laravel → Réponse JSON → Affichage

---

## 🌍 Multilangue (i18next)

- Support Français 🇫🇷  
- Support Arabe 🇲🇦  
- Traduction dynamique avec `react-i18next`  
- Gestion automatique RTL (Right-To-Left)
